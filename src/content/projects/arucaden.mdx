**arucaden.my.id** is my personal portfolio website, architected specifically to showcase my interactive projects, skills, and experiences.

### Executive Summary

Acting as the **sole Frontend Engineer & UI/UX Designer**, I designed this platform as a major upgrade from generic CMS templates. It functions as a blazing-fast static site while injecting deep javascript reactivity exactly where it's needed using Astro Islands and Svelte Runes.

### The Problem & The Solution

**The Problem:** Typical developer portfolios often feel like highly static corporate templates, or they swing in the complete opposite direction—relying on heavy client-side frameworks (like React) that bloat loading times and destroy SEO simply just to host a few buttons.

**The Solution:** I architected this website utilizing Astro for lightning-fast server-side HTML generation, Svelte 5 for precise, reactive "Islands" (like modals and overlays), and UnoCSS to completely eliminate CSS stylesheet bloat. The result achieved a unique cyber-glitch aesthetic that loads almost instantly.

### Technical Details

- **Blazing Fast Performance:** Powered entirely by Astro's static site generation and island architecture.
- **Reactive Components:** Leveraged Svelte 5's new `$state` Runes to drive high-performance overlays and reactive data bindings.
- **Utility-first Styling:** Completely custom-themed using UnoCSS for on-demand optimized CSS output.
- **Cyber Animations:** Integrated native JavaScript intersection observers, CSS transitions, and completely custom glitch effects.

### Challenges & Learnings

Merging Astro routing with Svelte's component hydration was extremely tricky at first. I learned deeply about partial hydration concepts (figuring out when to use `client:idle` versus `client:visible`). Furthermore, adapting to Svelte 5's explicit Runes forced me to understand exactly how reactivity flows, making my interactive components drastically more robust. The biggest takeaway? Always design with the end-user's navigation experience in mind over personal developer ego!
